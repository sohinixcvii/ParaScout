ParaScout documentation
=======================

ParaScout is a lightweight Python library for evaluating and visualising how
well a parameter space has been explored during simulation campaigns. It
accepts collections of NumPy arrays and dispatches each one to an appropriate
interactive `Plotly <https://plotly.com/python/>`_ visualisation based on the
array's shape.
